# LUCCA DEVISES

Back test to integrate LUCCA.


## Run Locally

Clone the project

```bash
  git clone https://github.com/Benoit2109/Lucca.git
```

Go to the project directory

```bash
  cd test-back-lucca
```

Install dependencies

```bash
  npm install
```

Running the program

```bash
  ./luccaDevises <nom du fichier>
```
